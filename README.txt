ArcHexagons

This script/model/workspace enables the creation of Hexagons to cover a specified extent. These hexagons can serve as planning units for prioritization exercises. 

The number of hexagons created can be specified by:
- Setting the width of a single hexagon
- Setting the number of hexagons desired in the E-W direction
- Setting the number of hexagons desired in the N-S direction
- Setting the total number of hexagons desired for the entire study area
